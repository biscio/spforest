\section{The data used for the examples}

This vignettes illustrate how to use spatial random forests to estimate the 
intensity of a point process. 
To that end, we will work with the data set \texttt{bei} from \spst, 
together with the covariates from the tropical forest data set. 
The covariates are loaded in the the package \pkg{sandbox} 
in the list \texttt{beisoil}.


\includegraphics{Intro-004}

Each element of the list \texttt{beisoil} is a \texttt{im} object from the package 
\pkg{spatstat-geom}. Roughly, this can be seen as a pixel images. 
In order to be able to work with this data, we need to transform them 
to have the same resolution. 
This have been done with the following code whose output is available in \texttt{beisoilres},

\begin{Schunk}
\begin{Sinput}
> zsoil <- NULL
> zsoil[[1]] <- beisoil[[1]]
> zsoil[[2]] <- beisoil[[2]]
> for (i in 3:15) {
+   W <- commonGrid(beisoil[[1]], beisoil[[i]])
+   zsoil[[i]] <- beisoil[[i]][W, drop = F]
+   rm(W)
+ }
> names(zsoil) <- c(
+   "elev", "grad", "Al", "B", "Ca",
+   "Cu", "Fe", "K", "Mg", "Mn",
+   "P", "Zn", "N", "N_min", "pH"
+ )
\end{Sinput}
\end{Schunk}

\section{Fitting a spatial intensity tree}

The function \texttt{treerec} allows to fit a spatial intensity tree 
to a point pattern. 

\begin{Schunk}
\begin{Sinput}
> inttree <- treerec(
+   X = spatstat.data::bei,
+   threshold = 30000,
+   score = "lcv2",
+   listcovariates = beisoilres,
+   mtry = 1,
+   tol = Inf,
+   minpts = 50,
+   minsplitq = 0.5,
+   maxsplitq = 0.5,
+   inforest=FALSE
+ )
\end{Sinput}
\end{Schunk}

The output is an object of class \texttt{sptree} define in the package \pkg{Rsandbox} 
together with methods to print, plot, summarise and predict the intensity in a given region.

\section{Fitting a spatial intensity forest}


\begin{Schunk}
\begin{Sinput}
> forest <- RforestPP(
+   X = spatstat.data::bei,
+   listcovariates = beisoilres,
+   score = "lcv2",
+   p = 1,
+   Ntree = 3,
+   threshold = 30000,
+   cores_trees = 1,
+   mtry = 1 / 3,
+   tol = Inf,
+   minpts = 50,
+   minsplitq = 0.5,
+   maxsplitq = 0.5
+ )
\end{Sinput}
\end{Schunk}
